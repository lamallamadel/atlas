Atlasia · design system

A quieter way
to do real estate.

A unified visual language for the Atlasia portal, marketing site, and Pro workspace — across iOS, Android, and Web. Marine + copper, editorial display type, restrained density.
01 · Foundation
Tokens
Color, type scale, spacing, radius, shadow. Light + dark.
02 · Building blocks
Components
Buttons, fields, badges, cards, tabs. Web defaults plus iOS / Android adaptations.
03 · In the wild
Screens
Auth, dashboard, dossiers, annonces, calendar — desktop, web mobile, iOS, Android.
Visual DNA
  • Marine #0d2c4a — chrome, headings, primary actions.
  • Copper #b5622e — italic display moments, key emphasis.
  • Warm off-white #f8f7f4 — pages, instead of cool grey.
  • Playfair Display for marketing/page titles. Italics for emotional emphasis.
  • Plus Jakarta Sans for UI labels, body, forms.
  • Restrained density. Generous spacing. No gradient slop.
Platform principle
Same language, native voice.
Shared component library. iOS uses Cupertino chrome (large titles, glass tab bars). Android uses Material 3 (top app bar, FAB, nav rail). Web sits between them — same colors, denser data tables.
